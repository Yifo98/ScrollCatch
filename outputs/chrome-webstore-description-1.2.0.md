# Chrome Web Store listing — ScrollCatch 1.2.0

## 中文说明

ScrollCatch — Full Page Capture 是一款本地优先的完整网页截图工具。它可以捕获长文章、在线文档、知识库、后台页面与网页报告，也支持飞书 / Lark 等在线文档中已经打开的 PPT / 演示稿逐页截取。

收其全貌，存其有度。

### 为什么安装 ScrollCatch

- 一键截取完整网页，也可以从当前位置继续向下截取或框选需要的范围。
- 超长页面会安全拆成连续分段，保留结束位置，并可回到原文继续截取下一段。
- 快速结果先帮助你检查截图是否完整；需要精细处理时，再进入统一编辑工作台。
- 支持自由裁切、精确数值、自定义分页线、多段排序与合并导出。
- 支持 A3、A4、A5、B5、Letter、Legal、Tabloid 纸张以及横版 / 竖版 PDF。
- 支持完整 PNG、JPEG、分页 PNG ZIP、分页 JPEG ZIP、单段 PDF 与多段合并 PDF。
- 飞书 / Lark PPT 预览页可按幻灯片逐页捕获，并按页导出图片或 PDF。
- 截图过程中可按 Esc 停止并保留已截内容；框选范围也可按 Esc 取消。
- 中英文界面可随时切换；默认 English，主动选择中文后会在本机记住。

### 更适合长内容的工作流

ScrollCatch 不会强行把极长网页塞进一张超高画布。到达单段安全上限后，它会保存当前结果与结束位置。你可以先确认衔接处，再回到原文继续下一段，最后在同一个编辑工作台中排序、裁切、分页并合并导出。

工作台为了保持流畅，会使用轻量预览；局部模糊属于正常现象，不影响最终导出。PNG、JPEG 和 PDF 导出会重新读取本地保存的原始截图切片。

### 隐私

ScrollCatch 采用本地优先设计。截图、编辑草稿和临时缓存保存在浏览器扩展的本地存储中，不会上传到外部服务器。扩展不包含账号系统、广告、统计分析或远程代码。

## English Description

ScrollCatch — Full Page Capture is a local-first browser extension for capturing complete web pages, long articles, online documents, knowledge bases, dashboards, and reports. It can also capture opened Feishu / Lark PPT and presentation previews slide by slide.

Capture the whole, preserve with restraint.

### Why install ScrollCatch

- Capture a complete page, continue downward from the current position, or select a custom range.
- Continue extremely long pages in linked sections while preserving the exact end position.
- Review captures quickly, then open one unified workbench only when you need precision editing.
- Crop freely, enter precise values, add custom page breaks, reorder sections, and combine exports.
- Export real A3, A4, A5, B5, Letter, Legal, and Tabloid PDF sizes in portrait or landscape.
- Export full PNG or JPEG images, paginated image ZIP files, individual PDFs, or one combined PDF.
- Capture opened Feishu / Lark presentation previews slide by slide and export by page.
- Press Esc to stop an active capture and keep completed content, or cancel range selection.
- Switch between English and Chinese; English is the default and an explicit Chinese choice is remembered locally.

### A workflow designed for long content

ScrollCatch does not force an extremely long page into one unsafe canvas. When a section reaches the practical limit, it saves the result and the end position. Review the seam, return to the source, continue with the next section, then reorder, crop, paginate, and combine everything in the same workbench.

The workbench uses lightweight previews to stay responsive. Temporary softness in the preview does not affect exports; PNG, JPEG, and PDF output is rendered again from the original locally stored capture slices.

### Privacy

ScrollCatch is local-first. Screenshots, editing drafts, and temporary cache remain in browser extension storage and are not uploaded to external servers. The extension contains no account system, advertising, analytics, or remote code.
