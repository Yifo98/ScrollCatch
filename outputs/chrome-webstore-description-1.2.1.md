# Chrome Web Store listing — ScrollCatch 1.2.1

## 中文说明

ScrollCatch · 收是一款本地优先的完整网页截图工具。它可以捕获长文章、在线文档、知识库、后台页面与网页报告，也支持飞书 / Lark 等在线文档中已经打开的 PPT / 演示稿逐页截取。

收其全貌，存其有度。

### 主要功能

- 截取完整网页、从当前位置继续向下截取，或框选需要的范围。
- 超长页面安全拆分为连续分段，并保留结束位置继续截图。
- 通过快速结果检查截图，再进入统一编辑工作台精细处理。
- 支持自由裁切、精确数值、自定义分页线、多段排序与合并导出。
- 支持 A3、A4、A5、B5、Letter、Legal、Tabloid 纸张及横版 / 竖版 PDF。
- 支持完整 PNG、JPEG、分页图片 ZIP、单段 PDF 与多段合并 PDF。
- 支持飞书 / Lark PPT 预览页逐页捕获和导出。
- 截图过程中可按 Esc 停止并保留已截内容；框选范围也可按 Esc 取消。
- 默认 English，可切换中文并在本机记住语言选择。

### 隐私

ScrollCatch · 收采用本地优先设计。截图、编辑草稿和临时缓存保存在浏览器扩展的本地存储中，不会上传到外部服务器。扩展不包含账号系统、广告、统计分析或远程代码。

## English Description

ScrollCatch · 收 is a local-first browser extension for full-page capture. It captures long articles, online documents, knowledge bases, dashboards, and reports, including slide-by-slide capture of opened Feishu / Lark presentations.

Capture the whole, preserve with restraint.

### Main features

- Capture a complete page, continue from the current position, or select a custom range.
- Continue extremely long pages in linked sections while preserving the exact end position.
- Review captures quickly, then open one unified workbench for precision editing.
- Crop freely, enter exact values, add custom page breaks, reorder sections, and combine exports.
- Export A3, A4, A5, B5, Letter, Legal, and Tabloid PDFs in portrait or landscape.
- Export PNG, JPEG, paginated image ZIP files, individual PDFs, or one combined PDF.
- Capture opened Feishu / Lark presentation previews slide by slide.
- Press Esc to stop an active capture and keep completed content, or cancel range selection.
- English is the default; Chinese can be selected and remembered locally.

### Privacy

ScrollCatch · 收 is local-first. Screenshots, editing drafts, and temporary cache remain in browser extension storage and are not uploaded to external servers. The extension contains no account system, advertising, analytics, or remote code.
