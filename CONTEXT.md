# ScrollCatch（原 XF FullPage Capture）产品语境

本语境用于区分已经稳定交付的版本与正在设计的后续版本，避免迭代工作模糊稳定发布和开发状态的边界。

## Language

**现行版**:
已经通过商店真实交付并继续可用的稳定版本。后续版本通过验收前，不得改变它的安装包、发布状态或既有行为。
_Avoid_: 旧版、废弃版

**后续版本**:
沿用现有产品和商店身份、正在独立开发与验证的下一次正式升级。它通过发布门槛后接替现行版，但不得抹除现行版的回退能力。
_Avoid_: 新产品、第二个扩展

**截图工具**:
以方便、可靠地取得网页截图为首要承诺的产品。合并、分页和导出编辑是截图完成后的增强能力，不应拖慢或掩盖主要截图流程。
_Avoid_: 长内容工作室、文档编辑器

**快速结果**:
截图完成后优先出现的轻量结果界面，只承担尽快显示预览、反馈处理状态和提供常用导出。它不预先加载高级编辑能力。
_Avoid_: 结果编辑器、完整工作台

**编辑工作台**:
用户主动进入的高级处理界面，承担裁切、分页、分节合并和缓存管理。它的加载与运行不得阻塞快速结果。
_Avoid_: 合并页、高级结果页

**渐进结果**:
截图处理尚未全部完成时即可查看和操作的结果状态。预览可以由轮廓、缩略内容逐步变清晰，但最终导出始终使用原始截图数据。
_Avoid_: 加载页、等待页、低清结果

**活动分节**:
编辑工作台中当前正在调整、因而拥有高清交互预览的截图分节。其他分节保持轻量待命，直到用户切换过去。
_Avoid_: 当前画布、已选图片
