# 采用 QIDU ScrollCatch 品牌体系

状态：已接受，取代 ADR-0006。

ScrollCatch 使用小夫于 2026-07-15 确认的最终方向板作为唯一品牌视觉基准。产品图标采用琥珀金圆角底板、白色卷轴主体与三个连续节点，表达完整网页、长页续接、分段整理与收束保存；界面采用暖白、琥珀金和炭黑，不使用青色或绿色主按钮。

工具栏所需的 16、32、48、128 px 图标必须由同一份 512 px 生产母版生成，不为小尺寸改换造型。弹窗、快速结果和编辑工作台统一使用 `ScrollCatch · 收`、`收其全貌，存其有度。 / Capture the whole, preserve with restraint.` 与 `A QIDU Utility`，但品牌层不能改变截图、续接、飞书 / PPT、裁切、分页、缓存和导出行为。

本决策只取代 ADR-0006 中的旧钴蓝 Caught Sections 图标方向；ADR-0002 的现有产品迭代与历史回退边界继续有效。
