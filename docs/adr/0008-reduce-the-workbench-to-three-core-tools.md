# 工作台只保留三个核心工具组

编辑工作台移除 `整理`、`精修`、`分页`、`导出` 四阶段导航，只保留 `自由裁切`、`自定义分页`、`导出` 三个始终可见的核心工具组。截图内容保持为最大视觉区域；分段选择、顺序和排除放在左侧分段栏，不再形成另一个工作台概念。

自由裁切支持移动、八向缩放和重新拉出任意保留区域，不需要的整段可直接排除。多段导出默认在导出前把当前段的裁切、分页和导出参数同步到全部选中段，再按左侧顺序合并 PDF。

快速结果和工作台默认勾选 `导出后自动删除本次截图缓存`，以减少长页截图长期占用浏览器存储；这个会删除原始切片的选项必须在主要导出区域直接可见，用户可以在希望保留后续编辑能力时取消勾选。未完成长页或已经形成连续关系的分段，在只导出当前段时必须保留缓存，避免续截和最终合并失效；最终从工作台合并导出后再清理全部选中分段。
