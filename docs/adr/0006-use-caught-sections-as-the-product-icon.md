# 使用 Caught Sections 作为产品图标

状态：已由 [ADR-0009](./0009-adopt-the-qidu-scrollcatch-brand-system.md) 取代。

产品图标采用小夫已确认的 A 方案「Caught Sections」：三个白色内容分段与右侧捕获钩共同表达分段截图、续接和收拢长页的产品能力，置于纯钴蓝圆角底板上；不使用青色或绿色按钮式强调。

Chrome 所需的 16、32、48、128 px 图标必须从同一个透明母版等比例导出，不再为小尺寸替换造型。这样可以保持商店、扩展管理页和浏览器工具栏之间的品牌一致性，并确保小尺寸仍能辨认出内容分段和捕获钩。
